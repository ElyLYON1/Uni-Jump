                                Présentation: MODULE IMAGE 

Le module image est un projet de création d'image.Il utilise pour cela 2 classes et la librairie STL2.  Les fichiers utilisé sont :


-Pixel.h /Pixel.cpp
-Image.h /Image.cpp
-mainTest.cpp
-mainExemple.cpp
-mainAffichage

Ils sont dans le dossier src.


Il y a un Makefile qui compile les 3 mains et qui produits 3 exécutable.
Nommées test (mainTest.cpp),exemple(mainExemple.cpp) et affichage (mainAffichage.cpp)

Ils sont dans le dossier bin.

Toutes les données créee sont dans le dossier data.
Un fichier de génération de documentation doxygen est fourni dans le dossier doc.
